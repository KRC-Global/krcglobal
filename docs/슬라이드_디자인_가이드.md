## 전체 디자인 컨셉

### 키워드
- **전문성**: 공공기관의 신뢰감
- **현대성**: 디지털 혁신을 보여주는 모던한 디자인
- **글로벌**: 세계로 뻗어나가는 역동성
- **명료성**: 정보를 명확하게 전달

### 디자인 원칙
1. **일관성**: 모든 슬라이드에서 동일한 디자인 시스템 적용
2. **가독성**: 텍스트와 배경의 충분한 대비
3. **계층 구조**: 정보의 우선순위를 시각적으로 표현
4. **공백 활용**: 여유로운 공간으로 가독성 향상
5. **브랜드 정체성**: 한국농어촌공사 CI와 조화

## 컬러 팔레트

### Primary Colors (주 색상)

**KRC 네이비 블루** - 신뢰, 전문성
- HEX: `#1A4B7C`
- RGB: (26, 75, 124)
- 사용: 헤더, 제목, 강조 요소

**KRC 라이트 블루** - 혁신, 디지털
- HEX: `#4A90E2`
- RGB: (74, 144, 226)
- 사용: 서브 타이틀, 버튼, 링크

**KRC 스카이 블루** - 글로벌, 확장
- HEX: `#7CB9E8`
- RGB: (124, 185, 232)
- 사용: 배경 그라디언트, 보조 요소

### Secondary Colors (보조 색상)

**아이보리** - 따뜻함, 안정감
- HEX: `#CEC0AC`
- RGB: (206, 192, 172)
- 사용: 배경, 카드 배경

**그린** - 농업, 성장
- HEX: `#5CB85C`
- RGB: (92, 184, 92)
- 사용: 성공 지표, 긍정적 데이터

### Accent Colors (강조 색상)

**오렌지** - 에너지, 혁신
- HEX: `#F0AD4E`
- RGB: (240, 173, 78)
- 사용: 중요 알림, CTA 버튼

**레드** - 주의, 긴급
- HEX: `#D9534F`
- RGB: (217, 83, 79)
- 사용: 경고, 위험 지표

### Neutral Colors (중립 색상)

**다크 그레이** - 본문 텍스트
- HEX: `#333333`
- RGB: (51, 51, 51)

**미디엄 그레이** - 서브 텍스트
- HEX: `#666666`
- RGB: (102, 102, 102)

**라이트 그레이** - 구분선, 배경
- HEX: `#E0E0E0`
- RGB: (224, 224, 224)

**화이트** - 배경, 카드
- HEX: `#FFFFFF`
- RGB: (255, 255, 255)

### 그라디언트

**Primary Gradient** (헤더, 표지)
```css
linear-gradient(135deg, #1A4B7C 0%, #4A90E2 100%)
```

**Background Gradient** (배경 강조)
```css
linear-gradient(135deg, #f8fafc 0%, #e0f2fe 100%)
```

**Card Gradient** (카드 배경)
```css
linear-gradient(135deg, #ffffff 0%, #f8f9ff 100%)
```

## 타이포그래피

### 폰트 패밀리

**Primary Font (한글)**
- Pretendard Variable
- 웹폰트: `https://cdn.jsdelivr.net/gh/orioncactus/pretendard/dist/web/variable/pretendardvariable.css`
- 특징: 가독성 우수, 모던한 느낌

**Secondary Font (영문/숫자)**
- Inter
- 웹폰트: `https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600;700&display=swap`
- 특징: 숫자 가독성 높음

### 폰트 크기 시스템

**Heading (제목)**
- H1 (표지): 48px / Bold (700)
- H2 (섹션 제목): 36px / Bold (700)
- H3 (슬라이드 제목): 32px / SemiBold (600)
- H4 (서브 제목): 24px / SemiBold (600)

**Body (본문)**
- Large: 20px / Regular (400)
- Medium: 18px / Regular (400)
- Small: 16px / Regular (400)
- Caption: 14px / Regular (400)

**Special (특수)**
- 통계 숫자: 48-64px / Bold (700)
- 레이블: 12px / Medium (500)

### 줄 간격 (Line Height)
- 제목: 1.2
- 본문: 1.6
- 캡션: 1.4

### 자간 (Letter Spacing)
- 제목: -0.02em
- 본문: 0em
- 레이블: 0.05em (uppercase)

## 아이콘 및 일러스트

### 아이콘 스타일
- **스타일**: Line icons (outline)
- **두께**: 2px stroke
- **크기**: 24px, 32px, 48px, 64px
- **컬러**: Primary Blue 또는 텍스트 컬러

### 아이콘 사용 예시
```
📊 대시보드 → Chart/Analytics icon
🌍 글로벌맵 → Globe icon
💼 사업관리 → Briefcase icon
📄 문서관리 → Document icon
👤 사용자관리 → User icon
🔒 보안 → Lock/Shield icon
```

### 일러스트 가이드
- **스타일**: Flat design, 미니멀
- **컬러**: 브랜드 컬러 팔레트 사용
- **용도**:
  - 개념 설명 (추상적 개념 시각화)
  - 프로세스 플로우
  - Before/After 비교

### 이미지 처리
- **실제 화면 캡처**:
  - 해상도: 300dpi
  - 경계선: 1px, #E0E0E0
  - 그림자: 0 4px 12px rgba(0,0,0,0.1)

- **목업**:
  - 디바이스 프레임 활용
  - 현실감 있는 배치

## 애니메이션 가이드

### 애니메이션 원칙
1. **의미 있는 움직임**: 사용자 주의를 중요 정보로 유도
2. **적절한 속도**: 너무 빠르거나 느리지 않게
3. **자연스러운 easing**: ease-out 사용
4. **일관성**: 같은 요소는 같은 애니메이션

### 추천 애니메이션

**슬라이드 전환**
- Fade (부드러운 전환)
- Push (섹션 전환)
- 시간: 0.5초

**요소 등장**
- Fade In + Slide Up
- 순차적 등장 (0.1초 간격)
- 시간: 0.3-0.5초

**강조 효과**
- Scale (1.0 → 1.05)
- Glow (그림자 추가)
- Color shift

**차트 애니메이션**
- 바 차트: 하단에서 상단으로 성장
- 파이 차트: 시계 방향 회전
- 라인 차트: 좌측에서 우측으로 그리기
- 시간: 1-1.5초

### 비활성화 권장
- 과도한 회전
- 바운스 효과
- 깜빡임
- 3D 효과

## 차트 및 그래프

### 차트 컬러 팔레트
```
색상 순서:
1. #4A90E2 (Primary Blue)
2. #5CB85C (Green)
3. #F0AD4E (Orange)
4. #7CB9E8 (Sky Blue)
5. #CEC0AC (Ivory)
6. #9B59B6 (Purple)
```

### 바 차트
- **바 너비**: 데이터 수에 따라 조정
- **간격**: 바 너비의 50%
- **레이블**: 상단 또는 바 내부
- **그리드**: 수평선만, 얇고 연한 색

### 파이 차트
- **최대 섹션**: 6개 (그 이상은 기타로 묶음)
- **레이블**: 외부 배치 + 퍼센트
- **강조**: 중요 섹션 약간 분리
- **3D 효과**: 사용하지 않음

### 라인 차트
- **선 두께**: 3px
- **포인트**: 5px 원형
- **영역**: 반투명 채우기 (opacity: 0.1)
- **그리드**: 수평선 + 수직선, 얇고 연한 색

### 테이블
- **헤더**: Primary Blue 배경, White 텍스트
- **행**: 번갈아가며 배경색 (White / #F8F9FA)
- **경계선**: 1px, #E0E0E0
- **패딩**: 상하 16px, 좌우 20px
- **정렬**: 숫자는 우측, 텍스트는 좌측
