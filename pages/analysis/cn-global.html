🗺️ 전세계 유출곡선번호(CN) 산정

SoilGrids WCS (토양) + ESA WorldCover 2021 (피복) 무료 공개 API 기반 — 전 세계 어디서나 유역 폴리곤을 그리면 CN을 자동 산정합니다.

CN 분석 중...
SoilGrids WCS에서 토양 데이터를 수집하고 있습니다. 네트워크 상태에 따라 1~3분이 소요될 수 있습니다.
📡 사용 데이터 소스
토양
SoilGrids WCS
ISRIC · 250m 해상도 · Clay/Sand 0-5cm → USDA 텍스처 → HSG
피복
ESA WorldCover 2021
AWS S3 · 10m 해상도 · 11개 피복 분류 (v2.0)
CN표
TR-55 (SCS, USDA)
AMC-II 기준 CN → AMC-III 변환 적용
✏️ 유역 폴리곤 입력
🖱️ 사용 방법 좌측 지도에서 폴리곤 도구를 선택한 후 유역 경계를 클릭하여 그립니다. 마지막 점에서 첫 점을 클릭하면 완성됩니다.
